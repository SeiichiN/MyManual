以下のような $a があるとする。

$a = 'おはよう';

これを以下の関数func1の中で表示するとしたら、どうするか？
-------------------------------------------------
$a = 'おはよう';

function func1 () {
    print $a;
}

func1();
-------------------------------------------------
これは、うまくいかない。
$a が 未定義なのである。(Undefined variable)

では、どうするか？

まず、func1 が引数を一つとり、それを func1 の中で
表示するという手が思い浮かぶ。
-------------------------------------------------
$a = 'おはよう';

function func1 ($p) { // 一応 $p としておく。
    print $p;         // $a でも全然ＯＫ。
}

func1($a);
-------------------------------------------------

もうひとつの手は、グローバル指定をすることである。
-------------------------------------------------
$a = 'おはよう';

function func1 () {
    global $a;
    print $a;
}

func1();
-------------------------------------------------
グローバル指定することで、グローバル領域の変数を参照
できる。

さらにもうひとつの手は、関数を無名関数に変えて、
use で指定することである。
-------------------------------------------------
$a = 'おはよう';

$func2 = function() use ($a) {
    print $a;
};

$func2();
-------------------------------------------------
